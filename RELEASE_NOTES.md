# Zephaniah v1.1.0 Release Notes

**Release Date:** February 2026

## Overview

Zephaniah v1.1.0 is a major update introducing archive download capabilities, an integrated library viewer, expanded institutional coverage, and a redesigned user experience.

## New Features

### Archive Downloads & Library Viewer
- **Persistent Archive Downloads**: Download archives directly within the app with auto-extraction support
- **Library Viewer**: Browse downloaded documents in an integrated library interface
- **Offline Access**: View previously downloaded documents without an internet connection

### Expanded Coverage
- **27 Supported Agencies**: Now covering institutions across 8 categories
- Updated archive sources including Internet Archive, Google Drive, and government databases

### User Experience
- **Custom App Icon**: New distinctive application icon
- **About Page**: MimikaStudio-style layout with application information
- **Improved Search**: Enhanced DuckDuckGo search integration

### Legal & Compliance
- **Privacy Policy**: Comprehensive privacy policy page
- **Terms of Service**: Clear terms of service documentation
- **GPLv3 License**: Switched to GNU General Public License v3

## Improvements

- Hardened runtime services for improved stability
- Fixed library page layout issues
- Added dataset model for better data organization
- Updated README with accurate archive sources and datasets

## Technical Details

- **Version**: 1.1.0 (build 2)
- **Platform**: macOS (Apple Silicon and Intel)
- **Framework**: Flutter 3.x
- **Minimum macOS**: See system requirements

## Installation

1. Download `Zephaniah-1.1.0-macos.dmg`
2. Open the DMG and drag Zephaniah to Applications
3. On first launch, right-click the app and select "Open" (macOS Gatekeeper bypass)

## Checksums

SHA256 checksums are provided in `Zephaniah-1.1.0-macos.dmg.sha256`

## Known Issues

- First launch requires Gatekeeper bypass (right-click > Open)
- Large archive downloads may take significant time depending on connection speed

## Upgrade Notes

This version can be installed alongside or replace v1.0.0. User data and preferences are preserved during upgrade.
